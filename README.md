# Cordinalyus Rent Filo

GitHub Pages için hazırlanmış hızlı yüklenen statik site.

## Yayınlama
1. Bu klasörü bir GitHub reposuna yükleyin.
2. Repo ayarlarından **Pages** bölümünü açın.
3. Source olarak **Deploy from a branch** seçin.
4. Branch olarak **main** ve klasör olarak **/(root)** seçin.
5. Birkaç dakika içinde site yayınlanır.

## Hız optimizasyonları
- Video dosyaları web için sıkıştırıldı.
- Büyük logo ve ağır görseller optimize edildi.
- Hero ve lokasyon videolarına poster eklendi.
- CSS preload ve script defer kullanıldı.
- Harici font çağrısı kaldırıldı.
